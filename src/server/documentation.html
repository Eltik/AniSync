API Documentation

GET /

Returns the API documentation root.

GET /stats

Fetches the statistics.

Response: JSON object containing the statistics data.

HTTP/1.1 200 OK
{
    "anime": 0,
    "manga": 0,
    "novels": 0,
    "unresolvedEvents": 0
}

POST /stats

Posts new statistics.

Response: JSON object containing the updated statistics data.

HTTP/1.1 200 OK
{
    "anime": 0,
    "manga": 0,
    "novels": 0,
    "unresolvedEvents": 0
}
